# Продвинутый Backend. Модуль 27. Авторизация и аутентификация.

Практическая работа

## Задание 27.6

- Реализована система регистрации которая позволять регистрироваться при помощи логин-email-пароль.
- Реализована система авторизации при помощи логин-пароль.
- Регистрация и авторизация производится с проверкой CSRF-токена.
- Реализована страница администрирования, на которой возможно просматривать
  добавлять, редактировать и удалять пользователей.
- Реализована система с ролями.
- Создана страница на которую нельзя попасть, пока пользователь не авторизован. 
  На этой странице отображается один абзац текста и одна картинка. 
  Текст виден всем авторизованным пользователям, картинка — только пользователям с ролью  "Пользователь VK"
- Реализована система записи логов неудачных попыток авторизации.
- Реализована возможность регистрации и авторизации через VK ID.
